# Python Project

A basic Python project with core functionality.

## Features

- Command-line interface
- Basic utility functions
- Modular project structure

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

│   └── utils.py         # Utility functions
